# Environment placeholders

Create a local `.env` file from this checklist. Do not commit real values.

Runtime:
- NODE_ENV
- LOG_LEVEL
- TZ

MongoDB:
- MONGODB_URI
- SOURCE_DB
- RESEARCH_DB

RapidAPI credentials:
- RAPIDAPI_KEYS
- RAPIDAPI_KEY_PRIMARY
- RAPIDAPI_KEY_SECONDARY
- RAPIDAPI_KEY_TERTIARY

Provider base URLs:
- RAPIDAPI_SPORTAPI7_BASE_URL
- RAPIDAPI_SOFASCORE_BASE_URL
- RAPIDAPI_SPORT_API_REAL_TIME_BASE_URL
- RAPIDAPI_SOFASCORE_SPORT_API_BASE_URL
- RAPIDAPI_SOFASPORT_BASE_URL
- SOFASCORE_PUBLIC_API_BASE_URL

Optional browser fallback:
- ENABLE_BROWSER_FALLBACK
- BROWSERLESS_URL

Job controls:
- MAX_PROVIDER_RETRIES
- HTTP_TIMEOUT_MS
- INGEST_CONCURRENCY
- DATASET_VERSION
- FEATURE_VERSION
